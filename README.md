# WeekFlow

Mobile-first calendar, email, tasks, and family noticeboard — one app.

## Local development (Windows / Mac / Linux)

```bash
npm install
npm run dev
```

Open http://localhost:5000

## Production build

```bash
npm run build
npm run start
```

## Replit

1. Create a new Repl → **Import from GitHub** → `Angi-leeds/weekflow`
2. Replit runs `npm run dev` automatically (port 5000)
3. Ask Replit Agent to attach **PostgreSQL** and **Object Storage** when you're ready for household sync (see `replit.md`)
